# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

リモートリポジトリ：自分の手元のパソコン内にある保管庫
ローカルリポジトリ：サーバー上（GitHubなど）にあるチーム共有の保管庫

## プッシュとマージの違いは何でしょうか？

プッシュ：自分のパソコンで記録した変更履歴を、インターネット上の共有サーバーにアップロード（送信）する操作。
マージ：別のブランチ（作業枝）で行った変更を、別のブランチ（メインなど）に合流・統合させる操作。

## コミットとプッシュの違い

コミット：自分のパソコン内（ローカルリポジトリ）に変更履歴を保存する操作
プッシュ：ローカルリポジトリに保存した履歴をインターネット上の共有サーバー（リモートリポジトリ）にアップロードし、チームメンバーに共有する操作

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

ローカルでマージ：コードレビューがないためマージまでの時間が短い
プルリクエストでマージ：コードレビューがありエラーの発生を極限まで発生させず安全にマージする
